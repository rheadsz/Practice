AUTONOMOUS  NAVIGATION  BOT

"CURIO"

During mu junior year, I worked on a project that required extensive research and detailed implementation. The goal was to develop an autonomous navigation bot "CURIO", for our college premises so that it can help prospective students and visitors to go about the campus. It is powered by Robot Operating System (ROS) and can detect objects and calculate the optimal path in order to reach the destination. This project was part of the Automation and Robotics club at my college. The project was a collective effort of the following teams:

  • Raspberry Pi Team
  • Robot Operating System Team
  • PCB Design Team
  • Embedded System Design Team
  • Motor and PID Team
  • Structural Design Team

I was part of the Robot Operating System team, which consisted of 6 people. My task was to develop python scripts for bot navigation. It involved mapping and turtle bot simulation.
