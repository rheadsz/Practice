HEALTH CARE APP

"MEDICO APP"

This Health Care App was created using Java and Sqlite as the database in 2021. This app had the provision for the user to set reminders to consume medicine as well as order medicines from the online store. In addition to this, the user could also view the covid status of states and countries. This app demonstrated the CRUD operations where the user could:

  • Create - add medicine name with timings
  • Read - view the reminders
  • Update - update the date or time
  • Delete - delete the reminder set
